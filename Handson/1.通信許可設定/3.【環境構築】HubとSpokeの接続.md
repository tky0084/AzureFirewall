## Hub環境との接続


作製したSpoke環境の仮想ネットワークを開き、「ピアリング」→「追加」を押下する
<img width="960" height="516" alt="image" src="https://github.com/user-attachments/assets/de79cd1e-036a-46a6-ad25-9dce658cbca1" /><br><br>

リモート仮想ネットワークの概要を以下の通り入力する<br>
※　相手先(Hub環境側)から見た設定です。<br>

| 項目 | 入力方法 |
| -- | -- |
| ピアリング リンクの名前 | ピアリング名<br>例：peer-hub-prod-toSpoke1-01 |
| サブスクリプション | 相手先(Hub環境)のサブスクリプション |
| 仮想ネットワーク | 相手先(Hub環境)の仮想ネットワーク名 |

<br>
リモート仮想ネットワーク ピアリングの設定を以下の通りとする

| 項目 | 入力方法 |
| -- | -- |
| 'vnet-hub-prod-network-01' に 'vnet-Spoke1-prod-network-01' へのアクセスを許可する | 〇 |
| 'vnet-Spoke1-prod-network-01' からのトラフィック転送の受信を 'vnet-hub-prod-network-01' に許可する' | 〇 |
| 'vnet-hub-prod-network-01' 内のゲートウェイまたはルート サーバーに 'vnet-Spoke1-prod-network-01' へのトラフィックの転送を許可する | × |
| 'vnet-Spoke1-prod-network-01' のリモート ゲートウェイまたはルート サーバーを使用するために 'vnet-hub-prod-network-01' を有効にする | × |

<br>
ローカル仮想ネットワークの概要<br>
※　Spoke環境から見た設定です

| 項目 | 入力方法 |
| -- | -- |
| ピアリング リンクの名前 | ピアリング名<br>例：peer-Spoke1-prod-toHub-01 |

<br>

ローカル仮想ネットワーク ピアリングの設定

| 項目 | 入力方法 |
| -- | -- |
| 'vnet-Spoke1-prod-network-01' に 'vnet-hub-prod-network-01' へのアクセスを許可する | 〇 |
| 'vnet-hub-prod-network-01' からのトラフィック転送の受信を 'vnet-Spoke1-prod-network-01' に許可する | × |
| 'vnet-Spoke1-prod-network-01' 内のゲートウェイまたはルート サーバーに 'vnet-hub-prod-network-01' へのトラフィックの転送を許可する | × |
| 'vnet-hub-prod-network-01' のリモート ゲートウェイまたはルート サーバーを使用するために 'vnet-Spoke1-prod-network-01' を有効にする | × |
