### AzureFireallのデプロイ

<details><summary>詳細を見る</summary><div>

Azure Portalで、「Firewalls」と検索する<br>
<img width="960" height="516" alt="image" src="https://github.com/user-attachments/assets/0f936461-19f8-4d38-86f8-7c54221b95bb" /><br><br>

「Azure Firewall」であることを確認して、作成を押下する<br>
<img width="962" height="516" alt="image" src="https://github.com/user-attachments/assets/60e98a5a-5823-4a05-91f6-7d8ec5f0e2e2" /><br><br>

基本タブにて、次のように入力する

| 項目 | 入力方法 |
|:---------|:---------|
| サブスクリプション | Hubのリソースグループが存在するサブスクリプション |
| リソース グループ | Hubのネットワークが存在するリソースグループ |
| 名前 | Azure Firewallの名前 |
| リージョン | 仮想ネットワークと同じリージョン |
| 可用性ゾーン | なし |
| ファイアウォール SKU | StandardかPremium |
| Firewall policy | 「Add new」で新規作成し、ポリシー名とAzureFirewallと同じSKUを選択する |
| 仮想ネットワークの選択 | 既存のものを使用 |
| 仮想ネットワーク | Hubのネットワーク |
| パブリック IP アドレス | 新規作成し、パブリックIPアドレス名を入力する|
| Firewall Management NIC | チェックを外す |<br>

<img width="960" height="516" alt="image" src="https://github.com/user-attachments/assets/183a3e39-f58d-4820-8527-bbc744331689" /><br><br>

確認及び作成で作成する<br>
<img width="960" height="516" alt="image" src="https://github.com/user-attachments/assets/a6c45ea3-aadc-4cf8-b78a-c31ae059002a" />
